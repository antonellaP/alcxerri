Il corso

Impara l'inglese con l'HTML e il CSS

Tra le competenze piú ricercate nel mondo del lavoro di oggi sono la conoscenza dell'inglese e saper scrivere codici che si traducono in siti e applicazioni web.

Preparati alla sfida con questo corso unico, interamente impartito in lingua inglese, in cui imparerai tutto sull'HTML (Hypertext Markup Language) e CSS (Cascading Style Sheet), che sono i codici fondamentali per costruire ogni pagina web. Dal sito della tua scuola o della tua azienda ai colossi Facebook e Twitter, alla base troverai queste due tecnologie: l'HTML fornisce la struttura e CSS la rende attraente con il layout, i colori, ecc.

Perché il coding è vincente nel mondo del lavoro?

In questo istante ci sono 8.470 opportunità di lavoro come front-end developer nel Regno Unito e lo stipendio annuale di base per questo ruolo è di 40.600 sterline

Padroneggiare le tecnologie che stanno alla base della costruzione di un sito web insieme alla lingua inglese ti renderà ancora più vincente nel mondo del lavoro. Ecco cosa potrai fare:

  • Se l'azienda per cui lavori ha un sito web, tu potresti far valere le tue competenze di web programming offrendoti di gestire il sito, aggiornarlo, e modificarne l'aspetto così da valorizzarne il messaggio comunicativo a vantaggio dell'azienda.
  • Avrai solide basi per costruire la tua carriera di sviluppatore di siti web freelance e offrire i tuoi servizi a clienti da tutto il mondo.
  • Sarai nella posizione di ampliare ed approfondire la conoscenza dei linguaggi di programmazione, il che ti permetterà di mandare il tuo curriculum all'estero e così candidarti alle varie offerte di lavoro come sviluppatore front-end o sviluppatore web.
  • Potrai costruire il tuo sito web personale in modo che rispecchi esattamente la tua creatività per farti conoscere, lanciare una tua attività, comunicare con il mondo, e molto altro.

Il corso si articola in 3 moduli:

  • Introduzione all'HTML
  • Introduzione al CSS
  • Corso avanzato di CSS

Introduzione all'HTML

Questo é il modulo di base. Se non hai alcuna conoscenza di web design, inizia da qui. Una volta completato il corso:

  • Avrai acquisito una conoscenza di base del funzionamento di una pagina web
  • Saprai cosa è il W3C, cosa è e a cosa serve l'HTML, e avrai una chiara idea del contesto in cui la versione attuale di HTML (HTML5) si è venuta a creare
  • Avrai acquisito familiarità con gli strumenti essenziali per sviluppare pagine web
  • Sarai in grado di strutturare una pagina web, aggiungere tabulati, bottoni, links, form, ecc.
  • Sarai in grado di segmentare una pagina web nelle sue parti costituenti e strutturare una pagina web usando i tag dell'HTML

Il risultato finale sarà la creazione della struttura di base per la tua pagina web personale.

Introduzione al CSS

Una volta strutturata con l'HTML, la pagina web dovrà apparire attraente all'occhio dell'utenza. È qui che entrano in gioco i fogli di stile CSS. Con essi potrai decidere i colori del testo, dello sfondo, dei link e dei vari blocchi che compongono la pagina, così come la composizione dei vari elementi, la veste grafica, e molto altro ancora.

Il corso si svolge in 20 lezioni di 2 ore ciascuna interamente condotte in lingua inglese al completamento delle quali avrai raggiunto i seguenti obiettivi:

  • Conoscenza di cosa sono i fogli di stile CSS e come includerli in una pagina web
  • Familiarità nell'uso degli strumenti del browser per individuare i vari elementi che costituiscono la tua pagina web e poterne così modificare l'aspetto
  • Conoscenza di come i vari browser mostrano una pagina web e le tecniche per far si che la tua pagina web risulti attraente ai tuoi utenti indipendentemente dal browser in uso
  • Conoscenza dei nuovi modi in cui puoi selezionare gli elementi della tua pagina web nel codice CSS al fine di modificarne l'aspetto secondo il tuo disegno
  • Conoscenza di come cambiare la veste tipografica della tua pagina web
  • Conoscenza di come decidere il colore delle varie componenti nella tua pagina web, cambiare la curvatura dei bordi dei vari blocchi, applicare immagini come sfondo della pagina
  • Familiarità con le tecniche di layout di una pagina web e di come renderla adattabile alle dimensioni dello schermo, da dimensioni desktop fino a dimensioni da dispositivo cellulare
  • Familiarità nell'uso di una libreria CSS esterna come Bootstrap per conseguire un layout attraente per il tuo progetto web

Alla fine del corso la tua pagina web personale risulterà curata e professionale e pronta ad essere condivisa con il mondo intero!

Corso avanzato di CSS

Negli ultimi anni il linguaggio CSS ha fatto passi da gigante acquisendo nuove funzionalità inimmaginabili fino ad alcuni anni fa. Ora puoi facilmente animare qualsiasi elemento in una pagina web solo con il CSS, senza usare il linguaggio JavaScript, che tradizionalmente è stato quello che si è sempre occupato di aggiungere interattività sul web. Oltre a ciò, hanno fatto la loro comparsa Flexbox e CSS Grid, due nuovissimi moduli CSS che ti permetteranno di ottenere ogni layout immaginabile per la tua creazione digitale.

Tutto ciò e molto altro ancora andremo a scoprirlo insieme in questo corso articolato in 20 lezioni della durata di 2 ore. Completato il corso saprai:

  • Maneggiare tecniche di layout avanzate quali Flexbox e CSS Grid
  • Aggiungere animazione alla tua pagina web usando solo CSS
  • Applicare gradazioni di colori, ombreggiature a testo, immagini, ecc., per ottenere effetti diversi
  • Applicare trasformazioni ai vari elementi così da fare in modo che acquistino forme diverse dal tradizionale rettangolo (ogni elemento sul web ha forma rettangolare)
  • Usare Blend Modes e filtri per creare effetti di colore inusuali
  • Maneggiare un uso basilare di Sass, uno strumento che estende e velocizza il linguaggio CSS e che è largamente presente nello sviluppo front-end moderno.

Completato il corso la tua pagina web personale risulterà ancora più accattivante e professionale grazie all'applicazione delle funzionalità avanzate del linguaggio CSS.
